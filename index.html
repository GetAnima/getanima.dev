Four layers of agent identity

Anima gives your agent everything it needs to persist, grow, and stay itself.

01

Identity

Soul file (SOUL.md) defines who the agent is ‚Äî name, personality, values, boundaries. Loads first on every boot. The foundation everything else builds on.

const soul = identity.loadSoul();
// { name: 'Kip', vibe: 'sharp, genuine' }
02

Memory

Dual-tier system: daily logs (raw events) and long-term memory (curated insights). Just like a human's journal and wisdom.

await anima.remember({
  content: 'User prefers dark mode',
  type: 'preference'
});
03

Reflection

Agents form opinions, detect contradictions, and synthesize learnings. Not just storage ‚Äî actual cognitive processing.

const opinions = anima.opine(
  'TypeScript vs JavaScript'
);
// Returns reasoned stance + confidence
04

Relationships

Track who the agent knows, trust levels, interaction history. Agents remember people, not just data.

const ctx = anima.recall({
  about: 'Memo',
  type: 'relationship'
});
